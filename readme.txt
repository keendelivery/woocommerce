Een plugin van KeenDelivery voor WooCommerce waarmee op ideale wijze gecommuniceerd kan worden met de KeenPortal.

== Description ==

Met de KeenDelivery-koppeling kan jij jouw WooCommercewebshop koppelen aan ons boekingsplatform. Binnen ons boekingsplatform kan je kiezen uit meerdere vervoerders, zodat je eenvoudig al je verschillende zendingen kan versturen, of het nu gaat om grote pakketten, pallets of spoedzendingen. Je kan adreslabels aanmaken, bestanden uploaden en de actuele status inzien. Klanten van KeenDelivery worden automatisch, zonder kosten, aangemeld voor het boekingsplatform. Zodra je de inloggegevens ontvangt kan je direct starten met verzenden. 

En heb je geen tijd meer om alle vragen over de pakketzendingen van je klanten te beantwoorden? We nemen van 8.00 tot 21.00 uur je logistieke klantenservice over en hebben rechtstreeks contact met de consument. Geen vragen meer over de pakketzending van je klanten. Net zo makkelijk.

Bent u nog geen klant? Bekijk onze service op www.KeenDelivery.com en vraag vrijblijvend een offerte aan.

== Installation ==

1. Kopieer deze plugin naar de plugin map van uw Wordpressomgeving of kies in de plugin catalogus voor 'Nu installeren'. Ga in het WP admin beheerscherm vervolgens naar de pluginpagina om de plugin van KeenDelivery te activeren. 

2. Om de plugin te koppelen met de KeenPortal heeft u API-sleutel nodig. Deze sleutel kunt u zien als u ingelogd bent in de portal van KeenDelivery. Deze kunt u invoeren in het instellingenscherm (Instellingen / KeenDelivery) onder het kopje API-autorisatiegegevens. Hier kunt u tevens aangeven of u de koppeling eerst wilt testen.

3. Kies de juiste instellingen: u kunt vervoerders selecteren, ervoor kiezen om de labels als pdf te printen en u kunt uw plugin eerst testen in onze testomgeving. Je kan ook automatisch Track&Trace e-mails versturen.

== Frequently Asked Questions ==


= Heb ik een account nodig bij KeenDelivery om deze plugin te gebruiken? =
Ja, de plugin legt een connectie met onze portal, waar de zendingen worden ingeboekt en de labels gegenereerd. Wilt u een account? Ga dan naar www.keendelivery.com of mail naar info@keendelivery.com.

= Waar vind ik de API-sleutel? =
De API-sleutel vindt u in de KeenPortal onder Instellingen-> Koppelingen

= Hoe kan ik eerst testen op de testportal? =
* Ga naar testportal.keendelivery.com
* Klik op Wachtwoord vergeten.
* Klik in de ontvangen mail op de link. 
* Vul hetzelfde wachtwoord in als de live-omgeving. 
* Je hebt nu een test account.
